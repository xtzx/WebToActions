# WebToActions

`WebToActions` 是一个“浏览器执行优先、网络证据驱动、支持人工审核和动作抽象”的本地自动化工具。

当前仓库处于开发准备阶段，已经完成产品方案、技术方案和开发步骤拆解，下一步按进度计划进入 **阶段 0：技术 Spike**。

- 技术文档：[docs/技术文档/文档索引.md](docs/技术文档/文档索引.md)

## 当前技术路线

- 后端：`Python 3.11+`
- API：`FastAPI`
- 前端：`React + TypeScript`
- 浏览器控制：`Playwright Python`
- 本地存储：`SQLite + 文件对象区`
- 长任务状态：`SSE`
